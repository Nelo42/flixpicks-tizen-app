# FlixPicks TV App

Samsung Tizen web app for browsing movies and TV shows from FlixPicks.net and controlling the FlixPicks MediaOS streaming box.

## Installation via TizenBrew

1. Open TizenBrew on your Samsung TV
2. Navigate to "Install from GitHub"
3. Enter: `gh/Nelo42/flixpicks-tizen-app`
4. Follow the installation prompts

## Features

- Browse movies and TV shows
- Search content
- View details and trailers
- Control FlixPicks MediaOS streaming box
- Watchlist management

## Requirements

- Samsung Smart TV with Tizen 2.3+ (2019+ models)
- TizenBrew installed on your TV
